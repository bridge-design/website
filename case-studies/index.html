Case Studies

Manychat Design System

Revamped Manychat's design system through extensive component architecture and documentation upgrades. The project delivered 16 new React components, accessibility-driven refactoring, Storybook 8 migration, and established a strong design system community. The collaboration between Bridge the Gap and Manychat's team resulted in a scalable foundation for form elements and an improved developer experience through ultra-defined components.

Yara International Design System

Refining Yara design system: in this project, we enriched a React library with more than 10 new components and refactored many others. We also conducted an accessibility audit, developed new tokens structure and migrated design system documentation to Storybook@7. Key efforts included infrastructure optimization, thorough documentation overhaul, and customizing the Storybook UI based on user feedback.

F-Secure's Design System Strategy

Exploring the collaborative journey of developing a design system strategy at F-Secure, highlighting the principles, roadmap, and tool recommendations aimed at fostering excellent digital experiences.

AppOmni Design System

Transformed AppOmni's user interfaces with an innovative design system. Web Components, tokens, comprehensive documentation, and seamless integration.

METRO Design System

Metro Design System is a multiperspective product for designers and developers that helps them create digital experience for millions of METRO customers in 22 countries.

Elisa Design System Renewal

Renewal of the design system at Elisa took place in the years 2020 and 2021. The the project included complete re-design and re-coding of 35 components and solid foundational work in design, development, infrastructure, and community managing areas.
