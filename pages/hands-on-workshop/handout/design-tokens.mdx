## What are design tokens

Design tokens are the values needed to construct and maintain a design system

- Colors
- Typography
- ...

## How to design with tokens

### Install Design Tokens plugin

1. [Design Tokens by Lukas Opperman](https://www.figma.com/community/plugin/888356646278934516/Design-Tokens)

### Reuse and extend Figma styles

Reuse the defined text styles, colors (Background, On background, primary). Potentially you can add more styles, such a secondary or support colors (Success, Info, Error).

### Export design token file

Export json file
